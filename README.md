# 👩🏻‍💻 Caio Victor

**`Programador`**

Meu nome é Caio Victor, tenho 22 anos e sou natural de Goiás. Concluí meu ensino médio integrado a Redes de Computadores no IFGO e atualmente estou no 7º periodo de Sistemas de Informação na Universidade Federal de Uberlândia (UFU). Durante minha trajetória na faculdade, participei de inúmeras Maratonas de Programação, o que me permitiu evoluir ainda mais meu raciocínio lógico e interpretação de problemas, além de pensar primeiramente em performance quando se trata de códigos. Esse GitHub é onde mantenho meus pequenos projetos que venho desenvolvendo ao longo do tempo. Nada demais, mas nada como um dia após o outro...
